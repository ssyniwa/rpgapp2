# 📝 README.md (Version 2 構成案)

## 🎮 AI生成RPGプロジェクト - Version 2: "Life in the Stats"

### 📌 プロジェクト概要
Geminiを活用して、キャラクター設定からスキルまでを動的に生成するRPG風アプリ制作編の第2弾です。
Version 2では、数値だけだった世界に**視覚的演出（アニメーション）**と**戦略的深み（ハクスラ・スキル数値化）**を加えました。

### 🚀 Version 2 の新機能
- **ハクスラ・ドロップシステム**: 敵を倒すと50％の確率で装備ドロップ。武器2本、防具1枚の3スロット制。
- **スキルの詳細数値化**: AIが生成したテキストに基づき、倍率（Power）、コスト（MP、種類を設定。
- **戦闘アニメーション**: 行動時の「発光エフェクト」と、被ダメージ時の「発光エフェクト」をTkinterで実装。
- **ランクシステム**: ステータス合計からS〜Gランクを自動判定し、レアリティに応じたカラーUIを表示。
- **永続化の強化**: 装備品や経験値、レベルアップ情報が正確にJSONへ保存・ロードされるようデータ整合性を改善。

### 🛠 技術的アップデート（主要な改善）
- **データ参照の最適化**: 
  - バトル画面とメイン画面での「参照（Reference）」を統一し、経験値やステータス上昇が即座に反映されるよう修正。
- **動的ステータス計算**: 
  - `base_atk`（基本値）と `equipments`（補正値）を分離。装備を外した際にステータスが正しく戻るロジックをプロパティ化。
- **UIコンポーネントの刷新**:
  - `ttk.Progressbar` によるHP/MPの可視化。
  - `Toplevel` ウィンドウを用いた詳細な装備・スキル確認画面。

### 🛠 セットアップ
1. **Geminiでキャラクター情報を作成**:
Geminiで以下のプロンプトを入力
   ```bash
   以下のJSON形式でキャラクター情報を作成
    {
    "name": "シオン", "job": "魔剣士", "hp": 120, "mp": 50, "atk": 22,
    "description": "闇の力を宿した剣を操る孤独な騎士。","image_path": "images/sion.png", "skills": [
    {
        "name": "ダークスラッシュ",
        "power": 12,
        "mp_cost": 10,
        "effect_type": "damage"
    }
    ]
    }
   ```
2. **ライブラリのインストール**:
   ```bash
   pip install pillow
   ```
3. **実行**:
   ```bash
   python rpgapp14.py
   ```
4. **メイン画面での入力**:
   json形式を入力部分にコピーし、敵か味方として登録
